CloudBees Eclipse Toolkit

Installation instructions

Use http://eclipse.cloudbees.com as an update site url in eclipse to install the toolkit

Prerequisites:

  • Eclipse 3.6 or newer
  • Subclipse (optional, forge SVN repository support)
  • Subversive (optional, forge SVN repository support). NOTE! Please make sure that subversive has a connector installed. If not then before installing add http://community.polarion.com/projects/subversive/download/eclipse/2.0/helios-site/ update site to your Eclipse installation.
  • eGit (optional, forge GIT repository support)
