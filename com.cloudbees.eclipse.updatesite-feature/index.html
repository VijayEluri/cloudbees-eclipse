CloudBees Eclipse Toolkit

Installation instructions

Use http://eclipse.cloudbees.com update site url in eclipse to install the toolkit.

Prerequisites

Eclipse 3.6 or 3.7

To use Forge SVN and GIT Integration (optional plugins):

  • Subclipse (at least one Subclipse connector must be available)
  • Subversive (at least one Subversive connector must be available, use http://community.polarion.com/projects/subversive/download/eclipse/2.0/helios-site/ as update site to install)
  • eGit

Make sure that the plugin itself is installed and configured before installing the Toolkit!! (Subclipse, Subversive or eGIT)

Do not install Subclipse and Subversive integration at the same installation.
